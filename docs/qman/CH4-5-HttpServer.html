HTTP Server

综述

工程 httpserv 使用 GNU microhttpd 提供一个轻量级的 HTTP Server 实现。为使用简单，适用于 Windows 平台的二进制文件已经包含在 C-- 源码包中；Linux 版本下需要自行安装所需的库。

概念模型

Cdec httpserv 是一个简单的 HTTP Server 实现。它提供了 Servlet 级别的功能支持，并支持自定义 URL 映射以优化 URL Dispatch 的速度。

Server 类是提供服务的主类，首先需要装配的是 URL 派发器，可以使用默认的 DefaultUrlDispatcher 类。

Windows 工程编译

重要：该工程目前不支持 X64 版本。

为使用简单，Cdec 代码库已包含了所需的二进制文件，位于 import-win\libmicrohttpd-0.9.17-w32 目录。该目录文件如下：

  • include 目录：头文件，由 httpserv.h 自动引用
  • lib 目录：LIB 文件，使用工程需要显式连接
  • bin 目录：DLL 文件，需要在运行时复制到应用程序目录（或 System32 目录）

原始的安装包位于此处，文件名 libmicrohttpd-0.9.17-w32.zip。该文件也位于代码库的 package 目录。

libmicrohttpd 二进制包还需要的 DLL 如下，为简单起见，这些文件也放入上述 bin 目录。

  • libiconv-2.dll
