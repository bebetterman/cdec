Curl Cdec

综述

工程 curlcdec 使用 libcurl 提供的 HTTP 等协议客户端实现。为使用简单，适用于 Windows 平台的源文件已经包含在 Cdec 源码包中；Linux 版本下需要自行安装所需的库（curl, curl-devel）。

概念模型

Curl Cdec 是对 libcurl 库的封装。主要用于实现 HTTP 客户端。

Windows 工程编译

重要：该工程目前不支持 X64 版本。

为使用简单，Cdec 代码库已包含了所需的二进制文件，位于 import-win\libmicrohttpd-0.9.17-w32 目录。该目录文件如下：

  • include 目录：头文件，由 httpserv.h 自动引用
  • lib 目录：LIB 文件，使用工程需要显式连接
  • bin 目录：DLL 文件，需要在运行时复制到应用程序目录（或 System32 目录）

原始的安装包位于此处，文件名 libmicrohttpd-0.9.17-w32.zip。该文件也位于代码库的 package 目录。

libmicrohttpd 二进制包还需要的 DLL 如下，为简单起见，这些文件也放入上述 bin 目录。

  • libiconv-2.dll
