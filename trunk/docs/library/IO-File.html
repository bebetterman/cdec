File Class (Core/IO)

File 类是一个静态类，用于对文件进行整体性操作。

Methods

static bool Exists(stringx path)

判断目标文件是否存在。

如果目标路径是一个有效的目录，此函数返回 false。

static bool Delete(stringx path)

删除目标文件。

static void Move(stringx pathOld, stringx pathNew)

将文件移动（改名）到目标路径。

如果两个路径不在同一个卷上，将抛出 IOException 异常。
