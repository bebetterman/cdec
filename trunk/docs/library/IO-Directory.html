Directory Class (Core/IO)

Directory 类是一个静态类，用于对目录进行整体性操作。

Methods

static bool Exists(stringx path)

判断目标目录是否存在。

如果目标路径是一个有效的文件，此函数返回 false。

static void Create(stringx path)

创建目标目录。

如果目标路径是一个有效目录，此函数不会抛出异常。

static void Delete(stringx path)

删除目标目录。

如果目标目录包含文件或子目录，此函数将抛出 IOException 异常。

static void Move(stringx pathOld, stringx pathNew)

将文件移动（改名）到目标路径。

如果两个路径不在同一个卷上，将抛出 IOException 异常。

ref<StringVector> GetDirectories(stringx path)

枚举指定目录下的所有子目录名。

ref<StringVector> GetFiles(stringx path)

枚举指定目录下的所有文件名。

ref<StringVector> GetFileSystemEntries(stringx path)

枚举指定目录下所有文件和子目录名。
